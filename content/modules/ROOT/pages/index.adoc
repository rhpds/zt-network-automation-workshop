= Ansible Network Automation Workshop

Welcome to the Ansible Network Automation Workshop!

== What you'll learn

In this workshop, you will:

* Explore the Ansible Automation Platform command-line tools and lab environment
* Write and execute Ansible Playbooks to automate network device configuration
* Gather and use Ansible facts from network devices
* Use network resource modules for vendor-agnostic configuration management
* Navigate and operate the Ansible Automation Platform web interface
* Create and run job templates for network automation tasks
* Build user-friendly surveys for job templates
* Configure Role Based Access Control (RBAC) for multi-user environments
* Design automation workflows with conditional logic and failure handling

== Who this is for

This workshop is designed for network engineers, systems administrators, and automation architects who want to learn how to automate network infrastructure using Red Hat Ansible Automation Platform.

**Experience level**: Beginner -- no prior Ansible experience required.

== Prerequisites

Before starting this workshop, you should have:

* Basic networking knowledge (routers, VLANs, SNMP)
* Familiarity with the Linux command line
* A web browser for accessing the lab environment

== Workshop environment

Your lab environment includes:

* An Ansible control node with `ansible-navigator` and a pre-built execution environment
* Four network devices: Cisco IOS-XE (rtr1), Arista EOS (rtr2, rtr4), and Juniper Junos (rtr3)
* An Ansible Automation Platform controller for the web-based exercises

All environment details and credentials are available in the lab interface tabs.

== Estimated time

This workshop takes approximately **6 hours** to complete.

You can work at your own pace.
